# m-h5-lab

## 项目简介

`m-h5-lab` 是一个用于归档和展示独立 HTML 页面的仓库。这里存放着多个彼此无关的 HTML 文件，每个文件均可独立运行，主要用于技术验证、原型演示或静态页面保存。

不针对特定域名或网站，仅作为页面文件的收集与版本管理用途。

## 目录结构

```
m-h5-lab/
├── index.html          # 仓库主页（可选）
├── pages/              # 存放各独立 HTML 页面
│   ├── page-001.html
│   ├── page-002.html
│   └── ...
└── README.md           # 本文件
```

- `pages/` 目录下每个 `.html` 文件为独立页面，可直接通过浏览器打开。
- 文件名通常反映页面内容或用途，但不构成任何外部承诺。

## 页面归档说明

- 所有页面均为静态 HTML，不依赖后端服务。
- 页面可能包含内联 CSS 和 JavaScript，或引用外部资源（如 CDN 库、字体等），需网络环境支持。
- 页面内容仅供个人学习、记录或演示，不代表任何正式产品或服务。

## 维护说明

- 本仓库不定期更新，新增或修改页面会通过 Git 提交记录。
- 欢迎通过 Issue 提交建议或报告问题，但请勿用于商业推广或无关请求。
- 不建议直接对生产环境引用本仓库中的页面地址，因为仓库结构或内容可能随时调整。
